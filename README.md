# -Smart-Classroom-Management-System
Smart Classroom Management System is a web-based application that helps teachers manage students, attendance, assignments, announcements, and classroom activities efficiently. The system provides a centralized platform where teachers can monitor student performance, track attendance, share study materials, and communicate with students.

Smart Classroom Management System

A web-based classroom management application that helps teachers organize student records, attendance, assignments, and announcements in one place.

Features

- Student Registration
- Teacher Dashboard
- Attendance Management
- Assignment Management
- Announcements
- Student Performance Tracking
- Search Students
- Responsive Design
- Data Storage using Local Storage

Technologies Used

- HTML
- CSS
- JavaScript

Installation

1. Download or clone the repository.
2. Open the project folder.
3. Run index.html in any browser.
